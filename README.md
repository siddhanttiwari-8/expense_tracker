Expense Tracker

Overview

Expense Tracker is a simple Python-based application that helps users record and manage their daily expenses. The project provides an easy-to-use menu-driven interface where users can add expenses, view all recorded transactions, calculate monthly spending, and analyze expenses by category. All expense data is stored in a CSV file, allowing the information to persist between program runs.

Features

- Add new expenses with date, category, description, and amount.
- View all recorded expenses.
- Calculate the total expenses for the current month.
- View category-wise expense totals.
- Store expense records in a CSV file for permanent storage.
- Simple command-line interface for easy interaction.

Technologies Used

- Python
- CSV Module
- Datetime Module
- File Handling

Project Structure

- "expense_tracker.py" – Main application source code.
- "expenses.csv" – Stores all expense records.

How to Run

1. Install Python 3 on your computer.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:
   python expense_tracker.py
5. Follow the on-screen menu to add and manage expenses.

Learning Outcomes

This project demonstrates the use of Python programming concepts such as file handling, functions, loops, conditional statements, CSV data management, and user interaction through a menu-driven application. It is suitable as a beginner-level project for showcasing Python fundamentals.
